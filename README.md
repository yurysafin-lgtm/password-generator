# 🔒 Advanced Password Generator

A sleek desktop application for generating strong and secure passwords, built with Python and CustomTkinter.

![Python](https://img.shields.io/badge/Python-3.14-blue) ![CustomTkinter](https://img.shields.io/badge/CustomTkinter-5.2-purple)

## ✨ Features

- Generate up to 5 passwords at once
- Adjustable password length (6–32 characters)
- Choose character types:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%)
- Password strength indicator
- One-click copy to clipboard
- Copy history (last 3 copied passwords)
- Clean dark UI

## 🚀 How to Run

1. Clone the repository:
   git clone https://github.com/yurysafin-lgtm/password-generator.git
2. Install dependencies:
   pip install customtkinter
3. Run the app:
   python ggenerator_pass.py
